There is no login page as account manager is not part of the application. The main tasks of this application will be with creating and editing playlist.This application will run on Linux itl computers. INSTALLATION------------ If you only want to view and run the code that created this app, open Netbeans and go to the PlaylistManager.java file in the fx package. Click the Run button to run the app.CONFIGURATION-------------By default, when the user opens the application the Now Playing list appears.Libraries/ApiThe main libraries we have used are Apcache Common, sqlite jdbc, Eclipselink and aws java sdk.
